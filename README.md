# Faisal's Portfolio — ReactBits Edition

A jaw-dropping developer portfolio powered by ReactBits-inspired animated components, Framer Motion, and GSAP.

## Features

- **SplitText** — 3D word-by-word entrance with blur + rotation
- **BlurText** — Multi-directional blur-to-focus reveal
- **GradientText** — Animated gradient text shimmer
- **RotatingText** — 3D flip role rotation
- **Aurora Background** — Floating gradient orb atmosphere
- **BeamsGrid** — Subtle grid + pulsing beam lines
- **SplashCursor** — GSAP-powered custom cursor with glow trail
- **Magnet** — Spring-physics magnetic hover on buttons/nav
- **TiltCard** — 3D perspective tilt with spotlight glow
- **SpotlightCard** — Cursor-following radial spotlight
- **CountUp** — GSAP-powered number counter
- **ScrollVelocityText** — Direction-aware marquee text
- **AnimatedList** — Staggered list entrance with blur
- **Dock** — macOS-style icon dock with proximity scaling
- **ScrollProgress** — Spring-physics top scroll bar
- **Floating Nav** — Centered pill nav with magnetic items

## Quick Start

```powershell
.\run.ps1
```

Or:

```powershell
npm install
npm run dev
```

Opens at **http://localhost:3000**

## Tech Stack

- React 18 + Vite 6
- Framer Motion 11
- GSAP 3.12
- Instrument Serif + Outfit + JetBrains Mono
- CSS Custom Properties

## Customize

Edit `src/App.jsx` for content, `src/components/ReactBitsComponents.jsx` for animations.
